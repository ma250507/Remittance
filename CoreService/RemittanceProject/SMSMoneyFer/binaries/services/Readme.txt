NCR MoneyFer System
===============
	System Prerequisites
		Software Prerequisites
			MS Windows 2003 server
			MS DOTNET Framework 2.0
			MS SQL Server Database Engine
	Installation Steps
		Database
			Copy database files (ldf, mdf) to HD
			Open MS SQL Server Enterprise and Attach database files
		Services
			Copy Services Binaries to HD 
			Use InstallUtil.exe 	to install included services 
	Configuration
		Open Config.xml for each service and edit it to correct values
